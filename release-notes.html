Premiere Voice AI Release Notes

Version: 1.1.0
Compatibility: Adobe Premiere Pro 26.0 and later
Architecture: UXP-based Extension

New Features

AI Voice Generation

  • Generate AI-powered voiceovers directly from text inside Adobe Premiere Pro.

Voice Customization

  • Gender selection
  • Multiple voice variants
  • Tone presets
  • Speed adjustment controls
  • Pitch adjustment controls

Custom File Naming

  • Specify custom filenames for generated audio files.

Import Options

  • Import generated audio directly to the active timeline.
  • Import generated audio to the project bin for later use.

Metadata Support

  • Store generation metadata alongside generated audio files.
  • Track original text, voice settings, speed, pitch, tone, and filename information.

Load Selected Clip

  • Restore generation settings from previously created Premiere Voice AI clips.

Modify Selected Clip

  • Regenerate existing AI-generated clips while preserving their placement in the timeline.

User Interface Improvements

  • Redesigned generation workflow.
  • Improved panel layout and organization.
  • Compact speed and pitch controls.
  • Enhanced status feedback and loading indicators.
  • Improved clip editing workflow.

Workflow Enhancements

  • Faster voice generation iteration process.
  • Simplified editing of previously generated clips.
  • Improved asset management and organization.
  • Metadata-driven clip restoration workflow.

Bug Fixes

  • Improved voice generation stability.
  • Improved timeline import reliability.
  • Improved metadata restoration behavior.
  • General usability and interface improvements.

Known Limitations

  • Audio preview functionality may vary depending on Premiere Pro and UXP environment capabilities.
  • Metadata restoration requires associated metadata files to remain available.

Upgrade Notes

This release introduces metadata-driven clip editing and restoration workflows. Generated clips created before metadata support was introduced may not be eligible for automatic settings restoration.

Note: Premiere Voice AI v1.1.0 requires Adobe Premiere Pro 26.0 or later.
