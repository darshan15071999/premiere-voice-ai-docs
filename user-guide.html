Premiere Voice AI User Guide

Premiere Voice AI is an Adobe Premiere Pro extension that enables AI-powered voice generation, editing, and direct integration into your Premiere Pro workflow.

Features

  • Generate AI voiceovers from text
  • Multiple voice and gender options
  • Tone selection presets
  • Speed and pitch controls
  • Custom filename support
  • Import generated audio directly to the timeline
  • Import generated audio to the project bin
  • Modify existing AI-generated clips
  • Restore generation settings from previously generated clips

Generating a Voiceover

1. Select a Gender.
2. Choose a Voice.
3. Select a Tone preset.
4. Enter a filename (optional).
5. Enter the text you want to convert into speech.
6. Adjust Speed and Pitch if required.
7. Click Generate Voice.

The plugin will generate an audio file using the selected settings.

Importing Audio

Import to Timeline

Click Import to Timeline to place the generated audio directly into the active sequence at the current playhead position.

Import to Bin

Click Import to Bin to add the generated audio to your Premiere Pro project without placing it on the timeline.

Editing Existing Voice Clips

Load Selected Clip

Select a previously generated Premiere Voice AI clip in your timeline and click Load Selected Clip.

The plugin will automatically restore:

  • Text
  • Voice
  • Gender
  • Tone
  • Filename
  • Speed
  • Pitch

This allows you to continue editing without manually re-entering settings.

Modify Selected Clip

After loading a clip:

  1. Update the text and/or voice settings.
  2. Click Modify Selected Clip.
  3. The selected clip will be regenerated using the updated settings.

Voice Controls

Tone

Tone presets help shape the delivery style of the generated voice.

  • Professional
  • Documentary
  • Voiceover
  • Dynamic
  • Casual
  • Comedic
  • Serious

Speed

Controls the speaking rate of the generated voice.

  • Lower values = slower narration
  • Higher values = faster narration

Pitch

Controls the pitch of the generated voice.

  • Negative values = deeper voice
  • Positive values = higher voice

Metadata & Project Management

Every generated voice file stores metadata that includes:

  • Original text
  • Voice selection
  • Gender
  • Tone
  • Speed
  • Pitch
  • Filename

This information enables the plugin to restore settings when editing previously generated clips.

Recommended Workflow

  1. Create your script.
  2. Select voice settings.
  3. Generate Voice.
  4. Import to Timeline.
  5. Review in your edit.
  6. Select the clip later if revisions are needed.
  7. Load Selected Clip.
  8. Modify the text or settings.
  9. Regenerate using Modify Selected Clip.

Troubleshooting

No Audio Generated

  • Ensure the backend service is running.
  • Verify that text has been entered.
  • Verify that a voice has been selected.
  • Ensure a Premiere Pro project is open.

Cannot Modify a Clip

  • Select a clip generated by Premiere Voice AI.
  • Ensure the media file still exists.
  • Ensure associated metadata files are available.

Settings Not Restoring

  • Verify the clip was originally generated by Premiere Voice AI.
  • Ensure metadata files have not been moved or deleted.
For best results, use clear punctuation and shorter paragraphs to produce more natural-sounding voiceovers.
